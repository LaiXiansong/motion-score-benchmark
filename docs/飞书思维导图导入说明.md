# 第 3 章综述思维导图 — 飞书导入说明

本目录提供 **第 3 章「文献综述」** 的总览思维导图，可直接导入飞书 **思维笔记**（导入后可在「大纲笔记 / 思维导图」视图间切换）。

## 文件

| 文件 | 格式 | 推荐 |
|------|------|------|
| `第3章综述-思维导图.opml` | OPML 2.0 大纲 | ✅ 优先尝试，结构最完整 |
| `第3章综述-思维导图.mm` | FreeMind 思维导图 | ✅ 备选 |

## 导入步骤（飞书云文档）

1. 打开 [飞书云文档](https://www.feishu.cn/) → **云文档** 或目标 **知识库**。
2. 点击 **新建** → **导入** → **导入为在线文档**（或：我的文档库 → 导入）。
3. 选择文件类型 **`.opml`** 或 **`.mm`**，上传对应文件。
4. 导入完成后打开文档，点击右上角 **思维导图** 图标，切换为导图视图。

> 飞书官方支持的思维笔记导入格式：`.xmind`、`.mm`、`.opml`（参见 [上传或导入本地文件](https://www.feishu.cn/hc/zh-CN/articles/360033241654)）。

## 导图结构概览

```
第3章 文献综述
├── 3.1 五种分类视角（A–E）
├── 3.2 多维度方法对比总表
├── 3.3 技术路线深述（几何/光流/学习/梯度）
├── 3.4 互补与张力
└── 3.5 LMA 启示
```

## 若导入失败

- 尝试另一种格式（`.opml` ↔ `.mm`）。
- 或使用飞书 **新建 → 思维笔记**，将 `第3章综述-思维导图.opml` 用任意文本编辑器打开，按层级手动粘贴大纲（Tab 缩进表示子节点）。
- 飞书 **画板** 不支持 XMind 直接导入；请使用 **思维笔记** 而非画板。
